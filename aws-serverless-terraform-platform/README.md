# aws-serverless-terraform-platform

Scaffold for an AWS serverless platform using Terraform.

Structure created:

- .github/workflows/deploy.yml
- terraform/environments/dev/{main.tf,variables.tf,terraform.tfvars}
- terraform/backend/{main.tf,variables.tf}
- terraform/modules/ (and service folders)
- frontend/
- backend/get/
- backend/put/
